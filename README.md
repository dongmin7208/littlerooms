## ❗Realization
- BEMによるフロントエンドの設計の理解
- いつでもデザイン変化に対応できる
- プロジェクトメンバーは、お互いのコードに貢献し合える
- メンバーが同じ言語でやりとりできるので、コミュニケーションが簡単
- 最初は構造設定に多くの時間を使用した。
- 現職者コードレビューを通じて協業過程を経験した。.
